How Sources Work

Watch URL: A brokerage or listing page checked daily for new content. The system detects page changes and extracts listing data. Example: https://www.colliers.com/en-ca/properties?market=edmonton&propertyType=multifamily

RSS Feed: An RSS/Atom feed URL from a brokerage or aggregator that publishes new listings. Example: https://broker-site.com/listings/feed.xml

Email Sender: An email address that sends listing alerts. The system parses those emails via IMAP. Example: noreply@realtor.ca

Folder Path: A local directory where broker packages (PDFs, docs) are dropped. The system extracts listing data from files. Default inbox: data/inbox/
