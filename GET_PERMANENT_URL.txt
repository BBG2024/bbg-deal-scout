OPTION C — Cloudflare Tunnel (Free, custom domain)
================================================
1. Go to https://dash.cloudflare.com and create a FREE account
   (no credit card required)
2. In the dashboard, go to: Zero Trust → Networks → Tunnels
3. Click "Create a tunnel"
4. Name it: bbg-deal-scout
5. Copy the tunnel token shown on screen
6. Create a file in this folder called:  .cloudflare_token
7. Paste the token into that file and save it
8. Run ENABLE_PUBLIC_ACCESS.bat — it will use your permanent URL

To connect a custom domain (e.g. deals.bluebeargroup.ca):
- In Cloudflare Tunnels, add a "Public Hostname"
- Domain: bluebeargroup.ca (add this to Cloudflare DNS)
- Subdomain: deals
- Service: http://localhost:8050
